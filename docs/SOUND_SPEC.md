# 사운드 에셋 명세서

## 필요한 사운드 목록

### 1. 명상 중 배경음 (water.mp3)
- **용도**: 명상 모드 진행 중 루프 재생
- **스타일**: 잔잔한 물소리, ASMR, 백색소음
- **길이**: 10~30초 (루프 가능하도록 끊김 없이)
- **포맷**: MP3, 128kbps 이상
- **볼륨**: 낮은 볼륨 (배경음)

### 2. 명상 성공 (bell.mp3)
- **용도**: 60초 명상 완료 시 1회 재생
- **스타일**: 맑은 종소리 또는 도트 게임 레벨업 효과음
- **길이**: 1~3초
- **포맷**: MP3

### 3. 명상/뽀모도로 실패 (fail.mp3)
- **용도**: 중도 포기 시 1회 재생
- **스타일**: 짧은 실패음, 도트 게임 miss 효과음
- **길이**: 0.5~1초
- **포맷**: MP3

### 4. 뽀모도로 완료 (pomodoro-done.mp3)
- **용도**: 뽀모도로 타이머 성공 시 1회 재생
- **스타일**: 밝은 팡파레, 도트 게임 스테이지 클리어 효과음
- **길이**: 2~4초
- **포맷**: MP3

### 5. 먹이 먹는 소리 (food-eat.mp3) — 향후 추가
- **용도**: 먹이 투하 후 금붕어가 먹을 때
- **스타일**: '뾱' 같은 귀여운 효과음
- **길이**: 0.3~0.5초
- **포맷**: MP3

## 생성 도구 추천
- **ElevenLabs SFX**: 텍스트로 효과음 생성
- **Suno AI**: 배경음 생성
- **Freesound.org**: 무료 효과음 다운로드

## 현재 상태
- 모든 사운드 파일이 더미이므로 제거됨
- 사운드 없이도 앱이 정상 동작하도록 구현 (try-catch)
- 사운드 파일이 추가되면 해당 경로에 배치하면 자동 적용
