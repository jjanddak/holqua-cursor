## 1. 프로젝트 개요

### 1.1 배경 및 목적
현대인의 고질적인 문제인 '둠스크롤링(Doomscrolling)'과 'ADHD성 집중력 저하'를 해결하기 위한 정서적 도파민 차단 앱. 단순히 기능을 강제하는 유틸리티를 넘어, 사용자와 금붕어 사이의 '정서적 유대감'을 통해 스스로 스마트폰을 내려놓게 만드는 것을 목표로 함.

### 1.2 핵심 가치
- **정지(Pause):** 무의식적인 폰 사용을 1분간의 명상으로 끊어냄
- **집중(Focus):** 금붕어와 함께하는 시간을 통해 업무 효율성 극대화
- **보상(Reward):** 시각적/청각적 피드백과 금붕어의 따뜻한(혹은 날카로운) 한마디

## 2. 사용자 경험

### 2.1 1분 명상 모드 (Mindful Pause)
1. **진입:** 앱 실행 후 수족관 화면 중앙을 꾹 누름 (Long Press)
2. **진행:** 60초 카운트다운. 터치 지점으로 금붕어가 다가와 맴돔. 백색소음(물소리) 재생
3. **성공:** 60초 완료 시 강한 진동과 함께 먹이 투하 애니메이션. 금붕어의 랜덤 메시지 출력
4. **실패:** 60초 전 손을 떼면 금붕어가 놀라 사라지며, 바닥에 "자리를 비울게요"라는 쪽지

### 2.2 금붕어 뽀모도로 (Dot-moro)
1. **설정:** 타이머 UI에서 집중 시간(20분~4시간) 설정 후 시작
2. **활성화:** 방해금지 모드 연동 안내 및 사운드 토글 선택
3. **유지:** 화면 켜져 있으나 10초 후 OLED 번인 방지 자동 밝기 저하 및 픽셀 시프팅
4. **성공:** 타이머 종료 시 대량 먹이 보상. 금붕어의 특별 칭찬 멘트
5. **실패:** 앱 이탈(5초 유예) 시 금붕어 가출, 수족관에 쪽지만 남음

## 3. 상세 기능 사양

### 3.1 메인 수족관 뷰 (The Tank)
- **단일 화면 원칙:** 복잡한 탭 없이 단 하나의 화면에서 모든 인터랙션 발생
- **금붕어 AI:** Reanimated 3로 화면 안 자연스러운 유영. 터치 시 가속도 붙여 이동
- **쪽지 시스템:** 실패 시 바닥에 도트 아이콘. 새 성공 전까지 영구 유지 → 심리적 부채감

### 3.2 집중 및 시스템 제어
- **Native Focus Mode:** 시스템 방해금지 모드 설정 유도
- **Burn-in Guard:** 10초 후 밝기 10% + 픽셀 시프팅
- **Ambient Sound:** 수족관 배경음 ON/OFF 스위치

## 4. 디자인 가이드라인

### 4.1 비주얼 컨셉: 16-bit Retro Pixel Art
- **레퍼런스:** 인세인 아쿠아리움(Insane Aquarium) 도트 감성 재해석
- **컬러:** 깊은 바다색(Dark Teal) 배경 — OLED 효율 + 눈 피로 감소
- **폰트:** 둥근모 스타일 오픈소스 도트 폰트

### 4.2 에셋 생성 전략
- **이미지:** AI 도구로 pixel art sprite sheet 생성 → Aseprite 편집
- **사운드:** AI 도구로 물방울, 먹이 먹는 소리, 배경음 생성
- **텍스트:** ChatGPT로 성격 부여된 50종+ 멘트 (quotes.json) 생성

## 5. 기술 스택

| 구분 | 기술 | 이유 |
|------|------|------|
| Framework | Expo (Managed) | 빠른 빌드, 네이티브 API |
| Animation | Reanimated 3 | UI 스레드 기반 끊김 없는 움직임 |
| State | Zustand | 가볍고 직관적 |
| Storage | react-native-mmkv | AsyncStorage 대비 압도적 I/O 속도 |
| Haptic | expo-haptics | 긴장감과 쾌감 전달 |
